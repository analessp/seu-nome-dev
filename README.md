# Seu Nome Dev

## Sobre o Projeto
**Seu Nome Dev** é uma aplicação web leve e divertida que gera um *nome de desenvolvedor(a)* único com base na data de nascimento do usuário.

A proposta é simples: combinar títulos do universo tech (como *Desenvolvedor(a)*, *Programador(a)*, *Sênior*) com sufixos criativos e bem-humorados (como *bugado(a)*, *das gambiarras*, *do commit bugado*), criando resultados únicos e compartilháveis.

---

## Funcionalidades

- **Entrada de Dados Interativa**  
  O usuário informa o dia e seleciona o mês de nascimento de forma simples e intuitiva.

- **Geração Automática de Nome**  
  Com um clique, o sistema processa os dados e cria um nome personalizado.

- **Validação Inteligente**  
  Mensagens amigáveis são exibidas para entradas inválidas (ex: dia 32, mês incompatível, etc.).

- **Exibição em Destaque**  
  O nome gerado aparece em uma área visualmente destacada.

- **Cópia com Um Clique**  
  Permite copiar o nome gerado rapidamente para a área de transferência.

- **Design Responsivo**  
  Interface adaptável para desktop, tablet e mobile.

- **Visual Moderno**  
  Interface com fundo animado, transparências e transições suaves.

## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias web padrão:

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização, responsividade e animações  
- **JavaScript (ES6+)** → Lógica, validações e interatividade
